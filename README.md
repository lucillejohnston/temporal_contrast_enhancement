# temporal_contrast_enhancement

### Offset analgesia / Onset Hyperalgesia
Codes the run the task using the QST.Labs TCSII with the 5 contact thermode and eVAS (as well as previous code using a Midi Sparrow 3x100 slider)

4 trial_types - hold_T1, step_up, step_down, and hold_T2
Period A = 5 seconds
Period B = 5 seconds
Period C = 20 seconds 

hold_T1:
A = T1
B = T1
C = T1

step_up:
A = T1
B = T2
C = T1

step_down:
A = T2
B = T1
C = T2

hold_T2:
A = T2
B = T2
C = T2
